And then we have the different lists, which are usefull when wanting to list things. There are two different types of lists, starting with un-ordered ones:

  • This is the first item
  • Item number two this is

And then there are ordered lists which hava a number in front of each item:

  1. This is the first item
  2. Item number two this is
